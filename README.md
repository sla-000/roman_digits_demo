Simple project with following features:

- AndroidX latest library
- MaterialButtons
- Service providing incrementing value
- 2 fragments: 
  - control buttons,
  - text with value
- Various animations (translation, rotate, scale, color) on touch, change value, and accelerometer data change
- Views binding with ButterKife
- MVP project structure
- Integer to Roman digits conversion
- Data exchange with Greenrobot EventBus
- Leak detection with LeakCanary
- Preferences save and load with Hawk
- Different layouts for portrait and landscape mode
- Translation to Russian locale
